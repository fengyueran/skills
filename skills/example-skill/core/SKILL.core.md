# example-skill core

## 目标
提供可跨编辑器复用的通用技能流程。

## 触发条件
- 用户明确提到该 skill。
- 任务意图与该 skill 范围匹配。

## 输入
- 必需：任务目标
- 可选：约束条件、目标文件、输出格式

## 通用流程
1. 明确目标与约束。
2. 仅加载必要参考资料。
3. 有脚本时优先执行 `scripts/`。
4. 产出结构化结果。
5. 给出可验证证据后再声明完成。

## 输出
- 结果摘要
- 改动文件清单（若有）
- 验证命令和结果

## 降级策略
- 若关键文件缺失或不可访问，说明阻塞点并提供最佳替代方案。
