# react-coding-standard（codex 适配）

在 Codex 中：

1. 在修改 React 代码前先读取 `../../references/react-coding-standard.md`。
2. 以 `../../core/SKILL.core.md` 作为任务工作流。
3. 评审时优先指出与规范对应的行为风险和正确性风险。
4. 除非用户要求更大范围的重构，否则保持修复方式与现有项目结构一致。
